# Online Quiz Application (Java)

## Project Description

The Online Quiz Application is a console-based Java program that conducts a multiple-choice quiz.
The program reads quiz questions from an external text file and displays them to the user one by one.
The user selects answers, and the system calculates the final score at the end of the quiz.

This project demonstrates how Java programs can read data dynamically from a file instead of hardcoding the questions in the program.


## Features

• Conducts a multiple-choice quiz in the console
• Reads quiz questions from an external file (`questions.txt`)
• Displays options and accepts user answers
• Automatically checks answers and calculates the score
• Easy to add new questions by editing the text file


## Technologies Used

• Java
• File Handling in Java
• Object-Oriented Programming (OOP)
• Scanner for user input

## How to Run the Project

1. Compile the Java program

javac QuizFromFile.java

2. Run the program

java QuizFromFile

3. Answer the quiz questions shown in the console.
